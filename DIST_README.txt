Before using this bot, you must create a Discord application
and bot account. Follow the below steps to do so:

 1. Go to https://discordapp.com/developers/applications/me
 2. Click on "New App" and give your bot a name
 3. Click on the "Bot" tab on the left
 4. Go down to "Privileged Gateway Intents" and enable "SERVER MEMBERS INTENT"
 5. Obtain your token by clicking "Reset Token" under the "Token" section
    near the top of the page
 7. Now navigate to the "Installation" tab on the left
 8. From this section, you can uncheck "User Install"
 9. Next, under the "Default Install Settings" > "Guild Install" section,
    add the "bot" scope
10. You should now see a new "Permissions" entry. Add the "Send Messages"
    permission
11. Finally, for the "Install Link" section, ensure it is set to
    "Discord Provided Link" and save your changes
12. Copy the link and paste it into your browser to add the bot to your server

Once this is complete, you can prepare to run the bot following the
following steps:

1. Run "CONFIGURE.exe" and follow the prompts
2. Once the configuration is complete, run "START.bat" to start the bot
